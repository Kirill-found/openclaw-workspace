## Le Nez — Hybrid AI подход (исправление картинок)

### Проблема
AI возвращал духи с неверными Fragrantica ID → показывались чужие бутылки.

### Решение: Hybrid подход (2 API вызова)
1. **analyzeVibeHybrid()** → AI извлекает mood/notes/keywords из фото
2. **NoteSearchService** ищет в локальной БД (bottles_labels.json — 24,102 духов)
3. AI выбирает топ-5 из найденных кандидатов с обоснованием

### Изменённые файлы
- `OpenAIService.swift` — добавлен `analyzeVibeHybrid`, структуры `AISelection`, `AIPick`
- `ScannerView.swift` — использует `analyzeVibeHybrid` вместо `analyzeImageAndRecommend`
- `TestAIView.swift` — аналогично
- `ResultView.swift` — убран fallback на AI fragranticaID, показывает placeholder если нет в БД
- `FavoritesView.swift` — убран fallback
- `PerfumeListView.swift` — убраны аккорды из UI (по запросу)

### Важные детали
- `NoteSearchService.SearchResult` НЕ имеет `rating` (убрал из кода)
- `AIRecommendedPerfume` init использует `bestSeason:` НЕ `season:`
- Духи без картинок: Wonderwood, Chergui, Tragedy of Lord George, Whispers in the Library, Clair de Musc, Like This
