## Le Nez — iOS приложение Кирилла

### Что это:
- Приложение для подбора парфюма по фото

- Фикс: `cd /usr/local/lib/node_modules/openclaw && sudo npm install sharp --include=optional && sudo npm rebuild sharp`

### Chrome extension OpenClaw:
- "Продукт недоступен" в Chrome Web Store
- Альтернатива: использовать profile="openclaw" вместо profile="chrome"
